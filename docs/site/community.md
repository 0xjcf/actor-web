# Community & support

Actor-Web is open source and under active development.

## Get involved

- **Source & issues** — [github.com/0xjcf/actor-web](https://github.com/0xjcf/actor-web).
  Bug reports and feature requests are welcome.
- **Companion project** — [ignite-element](https://github.com/0xjcf/ignite-element),
  the rendering layer Actor-Web sources plug into. See the
  [Ignite Element guide](/guides/ignite-element).

## Design docs

Architecture decisions and design rationale live alongside the code in the
repository's `docs/` directory — the topology/source DX, XState transition DX,
declarative subscriptions, and multi-process deployment designs. These are the
provenance behind the concepts documented here.

## Status

This documentation site and several runtime surfaces are still being built out.
Expect the Concepts, Guides, and API sections to grow; production-hardening items
are tracked as follow-ups (see [Production operations](/operations/production)).
